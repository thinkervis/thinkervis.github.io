CSV→Streamlit→지표 카드→공유

디버깅 기록

왜 배우나요?

이 단계부터 AI는 프로젝트를 확장하는 협업 도구로 사용합니다. 단, 직접 만든 그래프와 데이터를 바탕으로 개선을 요청합니다.

대시보드는 예쁜 화면이 아니라 사용자가 빠르게 상태를 이해하도록 돕는 화면입니다. 무엇을 먼저 보여줄지 정하는 것이 코드보다 중요할 수 있습니다.

친구가 처음 봐도 이해할 수 있는 제목, 축 이름, 단위, 상태 메시지를 넣습니다.

준비물

  • Raspberry Pi Pico 2 W + Grove Shield
  • 오늘 활동에 맞는 Grove 센서 또는 출력 모듈
  • USB 케이블, 노트북, 팀 기록지
  • CSV 양식 또는 활동 기록지

핵심 개념

오류 · 체크리스트

따라하기

  1. 직접 만든 데이터와 그래프를 준비합니다.
  2. AI에게 개선을 요청하되 조건과 원하는 출력 형식을 분명히 줍니다.
  3. Streamlit 템플릿에서 제목, 파일 경로, 열 이름을 우리 팀 데이터에 맞게 바꿉니다.
  4. 상태 메시지와 그래프가 사용자의 질문에 답하는지 확인합니다.
  5. 친구에게 링크나 화면을 보여주고 이해되는지 피드백을 받습니다.

실습 코드/데이터

import streamlit as st
import pandas as pd

st.set_page_config(page_title="우리 반 환경 대시보드", page_icon="🌈", layout="wide")
st.title("🌈 우리 반 환경 대시보드")

df = pd.read_csv("data/sample.csv")
st.dataframe(df)

col1, col2, col3 = st.columns(3)
col1.metric("평균 온도", f"{df['temp'].mean():.1f}℃")
col2.metric("평균 조도", f"{df['light'].mean():.0f}")
col3.metric("평균 소음", f"{df['sound'].mean():.0f}dB")

st.line_chart(df.set_index("timestamp")[["temp", "humidity"]])
st.bar_chart(df.groupby("place")[["light", "sound"]].mean())

장비와 라이브러리에 따라 함수명이나 핀 번호는 달라질 수 있습니다. 중요한 것은 구조를 읽고 우리 팀 상황에 맞게 고치는 것입니다.

바꿔보기 미션

  • 기준값, 색상, 메시지, 그래프 제목 중 하나를 우리 팀 상황에 맞게 바꿉니다.
  • 바꾼 전/후 결과를 사진, 표, 문장 중 하나로 기록합니다.
  • 친구 팀과 결과가 다르면 데이터나 기준값이 어떻게 다른지 설명합니다.
