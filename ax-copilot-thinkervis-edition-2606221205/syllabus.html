AX 실라부스 기준 학습 지도

이 페이지는 사용자가 제공한 두 실라부스 AX_standard와 AX_basic을 기준으로, 현재 교재의 어느 챕터를 읽고 가르치면 되는지 연결한 지도입니다. 강사는 이 페이지를 먼저 보고 수업 범위에 맞는 챕터로 이동하면 됩니다.

두 실라부스의 차이

AX_basicCopilot을 처음 쓰는 실무자용 A to Z 과정입니다. AI 모델/에이전트 개념, 접속법, PDF, Outlook, 주간보고, 프롬프트 빌더를 다룹니다.
AX_standard기초를 넘어 HTML MVP, 배포, PPT 구성, LG 스타일 프롬프트, Cowork/에이전트, 조별 실습까지 포함합니다.
현재 교재 반영 방향초보자 친절도는 유지하되, 표준 과정의 에이전트·배포·MVP 항목을 프로젝트와 HTML 챕터에 보강합니다.

아직 강사가 특히 챙겨야 할 표준 항목

  • HTML 배포: GitHub Pages는 외부 공개이므로 사내 데이터가 들어간 산출물은 올리지 않는다고 분명히 말합니다.
  • Outlook Copilot + Claude: 화면과 라이선스에 따라 모델 선택이 다를 수 있으므로 Copilot Web 대체 흐름을 같이 준비합니다.
  • Cowork/에이전트: 조직 설정에 따라 버튼이 안 보일 수 있습니다. 이때는 에이전트 설계서와 시스템 프롬프트 작성 실습으로 대체합니다.
  • LG 스타일 프롬프트: 현상, 원인, 대응, 기대효과와 표 형식을 요구하는 one-page 보고서 프롬프트를 반복합니다.
