버튼 감정등

오늘은 이런 일을 해요

오늘은 Grove Button + Grove RGB LED를 사용해 버튼 감정등을 만듭니다. 먼저 무엇을 어디에 꽂는지 확인하고, 짧은 코드를 실행해 장치가 실제로 반응하는 순간을 봅니다.

중요한 연결은 Button → D18, RGB LED → D16입니다. 이 포트 이름이 코드에도 그대로 등장하므로, 그림과 코드를 번갈아 보며 같은 이름을 찾으면 됩니다.

아직 AI는 사용하지 않습니다. 직접 실행하고, 값을 하나씩 바꾸고, 결과가 왜 달라졌는지 친구에게 설명하는 연습이 먼저입니다.

버튼 감정등입력 · 출력 · 조건문 · 상태감정등, 규칙표

연결 그림

그림 2-1정확한 연결 확인용

오늘은 Button → D18, RGB LED → D16만 기억하면 됩니다. 코드에서도 같은 포트 이름을 사용합니다.

따라하기

1. 포트 먼저 확인하기

Pico 2 W에 Grove Shield를 꽂고, Button → D18, RGB LED → D16 연결을 확인합니다. 코드를 보기 전에 포트 이름을 먼저 말해 봅니다.

2. 기본 코드 실행하기

제공된 코드를 그대로 실행합니다. 장치가 반응하면 연결과 실행 환경이 준비된 것입니다.

3. 숫자 하나만 바꾸기

색상값, 기준값, 기다리는 시간 중 하나만 바꿉니다. 한 번에 여러 개를 바꾸면 원인을 찾기 어렵습니다.

4. 친구와 결과 비교하기

친구 팀과 색, 속도, 기준이 다르면 코드의 어느 줄이 달랐는지 찾아봅니다.

5. 작동 원리 말하기

“입력은 ___이고, 처리는 ___이며, 출력은 ___이다” 문장으로 오늘 장치를 설명합니다.

실습 코드

아래 내용은 오늘의 출발점입니다. 그대로 따라 한 뒤, 우리 팀 상황에 맞게 표시된 부분을 바꿔 보세요.

from time import sleep
from grove_helper import GroveButton, GroveRgbLed

button = GroveButton(port="D18")
rgb_led = GroveRgbLed(port="D16")

calm_color = (40, 120, 255)      # 평소 색
happy_color = (255, 80, 160)     # 버튼을 눌렀을 때 색

while True:
    if button.is_pressed():
        rgb_led.set_color(happy_color)
    else:
        rgb_led.set_color(calm_color)
    sleep(0.05)

# 바꿔보기: calm_color와 happy_color 숫자를 우리 팀 색으로 바꿔 보세요.

아직 AI는 사용하지 않아요

이번 단계에서는 직접 실행하고, 직접 관찰하고, 직접 기록하는 경험이 먼저입니다. AI는 14차시부터 안전하게 사용하는 법을 배운 뒤에 사용합니다.

점검하기

  • 오늘 만든 것이 실제로 작동하거나 화면에 보이나요?
  • 내가 바꾼 부분을 한 문장으로 설명할 수 있나요?
  • 친구 팀과 결과가 다르면 그 이유를 말할 수 있나요?
  • Grove 모듈을 안내된 포트에 꽂았나요?
  • 아직 AI 없이, 내가 직접 실행하고 관찰한 결과를 기록했나요?
