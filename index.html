01차시40분입문

피코 LED 색 바꾸기

  1. 오늘 만들 것 보기

    완성 예시 이미지/GIF를 먼저 봅니다.

  2. 연결하기

    사진을 보고 Pico와 LED를 연결합니다.

  3. 코드 실행

    전체 코드를 제공하되 바꿀 부분을 강조합니다.

  4. 색 바꿔보기

    RGB 숫자를 수정해 우리 팀 컬러를 만듭니다.

  5. 공유하기

    친구에게 색의 의미와 바꾼 코드를 설명합니다.
