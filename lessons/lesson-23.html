대시보드 중간 공유

오늘은 이런 일을 해요

이제 직접 만든 데이터와 그래프를 바탕으로 대시보드 중간 공유을 진행합니다. 출발점은 언제나 여러분이 만든 CSV와 질문입니다.

AI는 화면을 더 보기 좋게 하거나 오류를 읽는 데 도울 수 있지만, 데이터의 의미와 기준값은 우리가 결정합니다.

대시보드는 예쁜 화면이 아니라, 처음 보는 친구가 현재 상태와 핵심 메시지를 빠르게 이해하도록 돕는 화면입니다.

공유 가능한 초안피드백 · 개선피드백 기록

오늘 흐름 그림

그림 23-1오늘 작업 흐름

장치를 새로 연결하기보다, 지금까지 만든 데이터와 기록을 화면·링크·발표로 정리합니다.

따라하기

1. 직접 만든 데이터 준비하기

AI에게 맡기기 전에 우리 팀 CSV와 그래프를 먼저 확인합니다.

2. 화면에 필요한 정보 고르기

평균, 최고값, 현재 상태, 장소별 비교 중 사용자에게 가장 필요한 것을 고릅니다.

3. 템플릿 수정하기

Streamlit 템플릿의 제목, 파일 경로, 열 이름부터 바꿉니다.

4. AI와 개선하기

그래프 색, 제목, 상태 문구처럼 작은 단위로 AI에게 개선을 요청합니다.

5. 친구에게 보여주기

처음 보는 친구가 화면을 보고 무엇을 알 수 있는지 확인합니다.

실습 코드

아래 내용은 오늘의 출발점입니다. 그대로 따라 한 뒤, 우리 팀 상황에 맞게 표시된 부분을 바꿔 보세요.

import streamlit as st
import pandas as pd

st.set_page_config(page_title="우리 반 환경 대시보드", page_icon="🌈")
st.title("🌈 우리 반 환경 대시보드")

df = pd.read_csv("data/classroom.csv")

st.subheader("원본 데이터")
st.dataframe(df)

st.metric("평균 밝기", f"{df['light'].mean():.0f}")
st.bar_chart(df.groupby("place")["light"].mean())

if df["light"].mean() < 400:
    st.warning("교실이 조금 어두운 편입니다.")
else:
    st.success("교실 밝기가 충분한 편입니다.")

# 바꿔보기: light를 sound, temp 같은 우리 팀 열 이름으로 바꿔 봅니다.

AI와 함께 개선하기

AI는 처음부터 대신 만드는 도구가 아니라, 여러분이 만든 코드와 데이터를 더 읽기 쉽고 보기 좋게 다듬는 도구입니다. 작은 단위로 요청하고, 결과를 반드시 실행해 확인하세요.

내 CSV에는 team, timestamp, place, light 열이 있어. 이 데이터를 사용해서 장소별 평균 밝기 그래프를 더 보기 좋게 개선해줘. 없는 열 이름은 만들지 말고, 바뀐 코드에는 주석을 달아줘.
