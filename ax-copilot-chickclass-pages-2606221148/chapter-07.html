챕터 7. 강사 지도서 — 수업 전날부터 마무리까지

좋은 자료가 있어도 강사가 준비하지 않으면 수업은 흔들립니다. 이 장은 강사가 수업 전날, 수업 시작, 실습 중, 마무리에서 무엇을 말하고 확인해야 하는지 정리한 지도서입니다.

수업 전날 체크리스트

  • 강의 계정으로 Copilot 접속 확인
  • 파일 첨부 가능 여부 확인
  • Outlook, Excel, Word 안 Copilot 버튼 표시 여부 확인
  • 첨부가 안 될 때 사용할 붙여넣기용 샘플 준비
  • 개인정보가 없는 샘플 데이터 준비
  • 화면 공유 시 글자 크기 확인
  • 인터넷이 느릴 때 설명으로 대체할 예비 흐름 준비
잠깐만요 │ 강사는 실패 장면도 준비해야 합니다

Copilot이 예상과 다른 답을 할 때가 있습니다. 이때 당황해서 새로고침만 반복하면 수강생도 불안해집니다. 오히려 ‘좋습니다. 이런 답이 나올 때 어떻게 고치는지 보겠습니다’라고 말하면 훌륭한 학습 장면이 됩니다.

마무리 과제

수업 끝에는 “오늘 배운 기능”을 묻지 말고 “내일부터 바꿔볼 업무 하나”를 쓰게 해야 합니다. 예를 들어 주간보고 초안, 회의록 정리, 고객 문의 분류, 교육 안내문 작성처럼 작고 반복되는 일이 좋습니다.

내일부터 Copilot으로 바꿔볼 업무 하나를 정리해줘.
1. 업무 이름
2. 지금 걸리는 시간
3. Copilot에게 맡길 단계
4. 사람이 반드시 확인할 단계
5. 첫 프롬프트 초안
형식은 표로 작성해줘.
강사 “오늘의 목표는 Copilot 기능을 많이 아는 것이 아니라, 내 업무 하나를 Copilot과 함께 시작할 수 있게 되는 것입니다.”
수강생 예상 “오늘 배운 걸 다 기억 못 할 것 같아요.”
강사 답변 “괜찮습니다. 전부 기억할 필요 없습니다. 내 업무 하나, 프롬프트 템플릿 하나만 가져가도 성공입니다.”
