버튼 감정등

오늘은 이런 일을 해요

오늘은 먼저 눈에 보이는 결과부터 만듭니다. 버튼 감정등 활동에서는 코드의 작은 숫자와 문장을 바꿔 보며 피코가 실제로 반응하는 순간을 확인합니다.

처음부터 모든 문법을 외우려고 하지 않아도 됩니다. 중요한 것은 ‘내가 무엇을 바꿨고, 결과가 어떻게 달라졌는지’를 말할 수 있는 것입니다.

AI는 아직 사용하지 않습니다. 여러분의 손으로 코드를 실행하고, 바꿔 보고, 친구와 비교하는 경험이 먼저입니다.

버튼 감정등입력 · 출력 · 조건문 · 상태감정등, 규칙표

오늘 한눈에 보기

따라하기

1. 먼저 실행해 보기

완성 예시를 보고, 기본 코드를 실행해 장치가 반응하는지 확인합니다. 오래 설명을 듣기보다 먼저 작은 성공을 경험합니다.

2. 한 곳만 바꿔 보기

색상값, 기다리는 시간, 메시지 중 하나만 바꿉니다. 한 번에 많이 바꾸면 무엇이 원인인지 알기 어렵습니다.

3. 결과를 말로 설명하기

“내가 바꾼 것은 ___이고, 그래서 결과가 ___로 달라졌다”라고 말해 봅니다.

4. 친구와 비교하기

친구 팀과 색, 속도, 기준이 다르면 코드의 어느 부분이 달랐는지 찾아봅니다.

5. 기록 남기기

오늘 바꾼 코드와 결과를 짧게 기록합니다. 이 기록이 다음 차시의 출발점이 됩니다.

실습 코드

아래 내용은 오늘의 출발점입니다. 그대로 실행하거나 복사한 뒤, 표시된 부분을 우리 팀 상황에 맞게 바꿔 보세요.

from machine import Pin
from time import sleep

button = Pin(15, Pin.IN, Pin.PULL_DOWN)
led = Pin("LED", Pin.OUT)

while True:
    if button.value() == 1:
        led.on()
    else:
        led.off()
    sleep(0.05)

아직 AI는 사용하지 않아요

이번 단계에서는 직접 실행하고, 직접 관찰하고, 직접 기록하는 경험이 먼저입니다. AI는 14차시부터 안전하게 사용하는 법을 배운 뒤에 사용합니다.

점검하기

  • 오늘 만든 것이 실제로 작동하거나 화면에 보이나요?
  • 내가 바꾼 부분을 한 문장으로 설명할 수 있나요?
  • 친구 팀과 결과가 다르면 그 이유를 말할 수 있나요?
  • 데이터를 다뤘다면 시간, 장소, 조건이 함께 기록되어 있나요?
  • 아직 AI 없이, 내가 직접 실행하고 관찰한 결과를 기록했나요?
