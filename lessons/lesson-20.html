20차시 · 바이브코딩·대시보드

대시보드란 무엇인가

오늘은 이런 일을 해요

이제 직접 만든 데이터와 그래프를 바탕으로 대시보드를 만듭니다. 대시보드란 무엇인가에서는 AI를 도구로 사용할 수 있지만, 출발점은 여러분이 만든 데이터입니다.

AI에게 처음부터 다 맡기지 않습니다. 먼저 우리가 가진 CSV, 그래프, 기준값을 확인하고, 그 다음 더 보기 좋게 바꾸는 데 AI의 도움을 받습니다.

대시보드는 예쁜 화면이 아니라, 친구가 한눈에 현재 상태와 핵심 메시지를 이해하도록 돕는 화면입니다.

대시보드 스케치지표 · 사용자 · 스토리화면 스케치

오늘 한눈에 보기

따라하기

1. 직접 만든 데이터 준비하기

AI에게 맡기기 전에 우리 팀 CSV와 그래프를 먼저 확인합니다.

2. 화면에 필요한 정보 고르기

평균, 최고값, 현재 상태, 장소별 비교 중 사용자에게 가장 필요한 것을 고릅니다.

3. 템플릿 수정하기

Streamlit 템플릿의 제목, 파일 경로, 열 이름부터 바꿉니다.

4. AI와 개선하기

그래프 색, 제목, 상태 문구처럼 작은 단위로 AI에게 개선을 요청합니다.

5. 친구에게 보여주기

처음 보는 친구가 화면을 보고 무엇을 알 수 있는지 확인합니다.

실습 코드

아래 내용은 오늘의 출발점입니다. 그대로 실행하거나 복사한 뒤, 표시된 부분을 우리 팀 상황에 맞게 바꿔 보세요.

import streamlit as st
import pandas as pd

df = pd.read_csv("data/sample.csv")

st.title("우리 반 환경 대시보드")
st.dataframe(df)

st.metric("평균 온도", f"{df['temp'].mean():.1f}℃")
st.line_chart(df.set_index("timestamp")["temp"])

AI와 함께 개선하기

AI는 여기서 처음부터 대신 만드는 도구가 아니라, 여러분이 만든 코드와 데이터를 더 읽기 쉽고 보기 좋게 다듬는 도구입니다. 작은 단위로 요청하고, 결과를 반드시 실행해 확인하세요.

내 CSV에는 team, timestamp, place, temp, humidity, light, sound 열이 있어. 이 데이터를 사용해서 현재 상태를 보여주는 Streamlit 지표 카드와 그래프를 더 보기 좋게 개선해줘. 바뀐 코드에는 주석을 달아줘.

점검하기

  • 오늘 만든 것이 실제로 작동하거나 화면에 보이나요?
  • 내가 바꾼 부분을 한 문장으로 설명할 수 있나요?
  • 친구 팀과 결과가 다르면 그 이유를 말할 수 있나요?
  • 데이터를 다뤘다면 시간, 장소, 조건이 함께 기록되어 있나요?
  • AI를 사용했다면 답을 그대로 붙이지 않고 검증했나요?
